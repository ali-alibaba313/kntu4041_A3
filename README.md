# Professional WebGIS Flask Project

Secure WebGIS with:

- Flask
- SQLite DB
- Password Hashing
- Flask-Login
- CSRF Protection
- OpenLayers
- GeoServer WMS + GetFeatureInfo

## Run in Cursor

### Create venv
python -m venv venv
venv\Scripts\activate

### Install deps
pip install -r requirements.txt

### Run
python app.py

Open:
http://127.0.0.1:5000
